# Python File Handling System
A simple menu-driven file handling system built using Python.

## Features
- Create files
- Read files
- Update files
- Delete files
- Exception handling
- Create Folder
- Rename File
- Delete Folder
- Create file in folder
## Technologies Used
- Python
- pathlib
- os module
